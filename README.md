# SmartCampus
## Gachon SLB
#### : Location based Android application	using Bluetooth beacon in Gachon University      

***

### **Brief Description**
#### **Primary Function**:
Attendance Check based BLE(Bluetooth Low Energy) using Raspberry Pi
#### **Supplementary Function**:
- Find the Classroom
- Find the Friend
- School Shuttle Bus
